# Meal Images

This directory contains images of our favorite meals.

## How to Add Images

1. Add your meal photos to this directory
2. Use descriptive filenames (e.g., `pizza.jpg`, `tacos.jpg`, `lasagna.jpg`)
3. Update the `imageFiles` array in `view-images.html` to include your new image filenames
4. Open `view-images.html` in your browser to see the gallery

## Supported Formats

- JPG/JPEG
- PNG
- GIF
- WebP
- SVG

## Tips

- Use high-quality photos for better viewing experience
- Consider resizing large images to reduce file size
- Use consistent naming conventions (lowercase, hyphen-separated)
