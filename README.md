# meals
We will store meal ideas

## Features

### Meal List
- `meals.txt` - A simple text file containing our favorite meals
- `suggestions.txt` - Additional meal suggestions

### Image Gallery
- `images/` - Directory for storing meal photos
- `view-images.html` - Interactive gallery to view all meal images

## How to Use the Image Gallery

1. **View images**: Open `view-images.html` in your web browser
2. **Add images temporarily**: Click "Select Images" button to browse and display images from your computer
3. **Save images permanently**: 
   - Add image files to the `images/` directory
   - Update the `imageFiles` array in `view-images.html` with your new filenames
   - Commit and push the changes to the repository

The gallery features:
- Grid layout that adapts to screen size
- Click images to view full-size
- Easy image selection from your computer
- Clean, responsive design
